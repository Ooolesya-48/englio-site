--- ФАЗА 4: Составь пары ---
Статус: ГОТОВО

Файлы:
- src/pages/PairsPage.tsx + .module.css

Функционал:
- Два столбца: слова слева, переводы справа (перемешаны)
- Нажимаешь слово → нажимаешь перевод (или наоборот) → проверка
- Правильно: зелёная вспышка + галочка + звук "пинг" + пара съезжает вниз (сжимается)
- Неправильно: красная тряска обеих плиток + звук ошибки
- Плитки растягиваются на всю высоту (flex: 1), угаданные сжимаются (flex: 0 0 32px)
- Звук финала — восходящее арпеджио C-E-G-C
- Звуки через Web Audio API (не файлы)

Настройки (шестерёнка в хедере, игра стартует сразу по дефолту):
- Количество пар: 5, 8, 10, 15
- Направление: EN→RU, RU→EN, Случайно
- Какие слова: Все, Слабые (<50 score), Новые (score=0)
- Выбор коллекции или отдельных слов
- Тоглы: Звуки вкл/выкл, Таймер показать/скрыть

Экран результата:
- Звёзды (0 ошибок = 3★, 1-2 = 2★, 3+ = 1★)
- Бонус за скорость (< 30 сек + ≤ 1 ошибки = +1 звезда)
- Кнопки: "Ещё раунд", "Настройки", "К играм"

Spaced Repetition: полноценное обновление (recognition + recall + difficulty + success_count + next_review)

- Тоглы: Звуки, Таймер

Экран результата:
- 3 звезды = "Безупречно!" (100%), 2 = "Хорошо!" (≥80%), 1 = "Попробуй ещё!"
- Статистика: правильно/всего, точность %, время
